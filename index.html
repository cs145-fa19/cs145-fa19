Data Management and Data Systems

Introduction to the use, design, and implementation of database and data-intensive systems.

Course Overview Overview

This course covers database design and the use of databases in applications, with a short introduction to the internals of relational database engines, the relational model, relational algebra, and SQL.

Students will also learn about database design and relational design principles based on dependencies and normal forms as well as many additional key database topics from the design and application-building perspective. This will include indexes, views, transactions, and integrity constraints. We'll also introduce MapReduce framework and key-value stores.

The class will culminate in a hands-on programming project in SQL+Python — a key part of the course — where you will query, visualize and predict from terabytes of data on BigQuery, a popular cloud database part of Google Cloud Platform.
